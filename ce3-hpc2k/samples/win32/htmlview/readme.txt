Sample: HTMLVIEW


The Htmlview sample allows the user to browse and open HTML (.htp) help files.
This sample code demonstrates the notifications and structures defined by 
Htmlctrl.h.


This sample code is not intended to be a full-featured browser, but a 
demonstration of the notifications and structures defined by Htmlctrl.h.  
As a result, there are many features of web design which are unsupported 
in this sample and may give unexpected results.
