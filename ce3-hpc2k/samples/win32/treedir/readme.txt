Abstract: TREEDIR is a small Windows application that highlights the use of:

1) Tree Control, and
2) FindFirstFile/FindNextFile.

The TREEDIR application allows you to browse the file system and display 
all directories and files by name.  TREEDIR makes use of OutputDebugString 
to provide feeback to the user while debugging.
