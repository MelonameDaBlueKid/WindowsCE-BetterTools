CEPad ReadMe
============
This sample is a one-form notepad application that demonstrates the use of the Commandbar control, the Common Dialog control, the File System control and the ImageList control. A resource file is used to store text strings for error messages and menu labels. The Commandbar control is initialized by the InitCommandBar() subroutine. The File System control is used to select the size, color and type of font and to open and save text files.  The ImageList control is used to place an icon on the command bar. 

To Install and Run the Sample
=============================
This sample runs without further modification.

Additional Notes
================
This sample runs in the emulator.

For information on the Commandbar control, the Common Dialog control, the File System control and the ImageList control see the Visual Basic for Windows CE online documentation.
