Scribble ReadMe
===============
This sample is a one-form drawing application that demonstrates the use of the Commandbar control, the Common Dialog control, the ImageList control and the PictureBox control.  The main drawing area is a PictureBox control.  Drawing events are handled by the PictureBox1_MouseMove subroutine which implements standard drawing methods.  The menu is hosted by the Commandbar control.  If the screen is not wide enough to correctly display the menu buttons bitmap, the Commandbar displays a standard text menu instead.  The ImageList control is used to hold the menu bar bitmap.  The Common Dialog control handles color and text selection.  This sample could be altered to support save file  and open file features using the Common Dialog control.

To Install and Run the Sample
=============================
This sample runs without further modification.

Additional Notes
================
This sample runs in the emulator.

For information on the Commandbar control, the Common Dialog control, the ImageList control and the PictureBox control see the Visual Basic for Windows CE online documentation.
