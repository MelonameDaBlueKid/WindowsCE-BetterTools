Sample Overview

Sample demonstrates how to implement an owner drawn ListView control.  Also demonstrates how to use various ListView notification events.

 

Source file listing and summary:

ceownerlistview.cpp – application entry point, owner drawn ListView message handler, window message handler

stdafx.cpp – precompiled type source file

newres.h – helper header for resource script

resource.h – header for resource script

stdafx.h – precompiled header

ceownerlistview.ico

ceownerlistview.rc – resource script

 

Requirements and Dependencies

This sample requires the Pocket PC 2003 or Smartphone 2003 SDK and Microsoft eMbedded Visual C++ to be installed. Prior to running the sample you may wish (if you haven’t already) to add some tasks, contacts, or appointments on your device (or emulator) so the sample app has something meaningful to display.

Topics

Owner drawn ListView control

ListView Notification messages.

Win32 drawing

GDI objects

Building the Sample

Load the project in eMbedded Visual C++, pick the target platform and SDK, and build normally.

 

Running the Sample

Run the sample from eVC (Shift-F5) or manually copy the executable to the device and run it from there.
