GetUUID Readme
===============
Both Pocket PC and Smartphone devices are required to contain a unique ID in the hardware of the device. This ID can be used for enterprise asset management systems or digital rights management, or other applications that require a device to be uniquely identified.

This sample works on either a real device or the emulator.  While the emulator contains a unique ID this value is simulated.

For additional information about the unique identifier, please consult the SDK documentation.
