Security Configuration Manager Powertoy for Windows Mobile
==========================================================

Security Configuration Manager Powertoy for Windows Mobile is a remote configuration tool
that runs on a host work station, that enables you to perform the following tasks:

    - Examine the security configuration of a Windows Mobile-based device.

    - Provision one of the Standard Security Configurations onto a mobile device.

    - Save a mobile device's security configuration.

    - Add a development certificate to a mobile device.

    - Sign a file with a Signing Certificate.

    - Check a file's digital signature.


To install the Security Configuration Manager Powertoy for Windows Mobile, just double-click 
on Setup.msi.
