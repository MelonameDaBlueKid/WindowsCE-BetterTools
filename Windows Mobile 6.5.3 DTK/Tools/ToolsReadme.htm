Windows Mobile 6.5.3 DTK - Tools

The Windows Mobile 6.5.3 DTK includes tools designed to help developers create applications for Windows Mobile devices.

GPS

Tools to help developers create location-aware applications.

Tool

Description

FakeGPS

FakeGPS enables developers to receive data using the GPS APIs even if there is no GPS receiver on the device. The GPS data is read from NMEA .txt files installed by default in \Program Files\FakeGPS\GPSFiles.

FakeGPS must run in privileged mode because it interacts with the GPS Intermediate Driver and writes to the privileged area of the registry.

FakeGPS ships with a two sample data files:

  • dixies.txt: NMEA files that immediately starts sending a GPS location to the GPS driver.
  • fakegpsdata.txt: Same as dixies.txt, but it takes some time to report a new location.

Default location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\GPS

GPSSettings

GPSSettings enables Windows Mobile Standard users to configure the GPS intermediate driver and works in the same way as the built-in GPS settings application on Windows Mobile Pro devices.

To use this tool, start GPSSettings.exe and set the hardware port to the port that receives the GPS data. If you are using a Bluetooth GPS receiver, make sure you pair the GPS device and set it to an outgoing port.

For example, if you choose COM7 as the outgoing port connected to your device on the Bluetooth settings, COM7 should be also the hardware port set on GPS settings. If you set a program port, it will receive a copy of the data streamed to the hardware port.

Default location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\GPS

Resolution Awareness

Windows Mobile devices are available in a variety of screen resolutions and orientations.

Tool

Description

SetDPI.exe

This tool is used to set the DPI of a bitmap (BMP) file.

Usage: setdpi <filename>.bmp <dpi>

Default location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Resolution Aware

HIDPI.exe and RES2EXE.exe

Used to mark a .Net Compact Framework application as resolution aware. For more information, see the MSDN topic Developing DPI Aware Applications.

Default location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Resolution Aware

Security

Several tools and sample security certificates are provided to help developers discover, simulate and predict issues regarding security and their applications.

Tool

Description

RapiSecurity

Consists of .CPF files and related provisioning XML files that allow changing the RAPI policy on a device.

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Security\RapiSecurity

DTK Development Certificates

These files contain .cer, .pfx., .pvk, and .spc files for Privileged, Unprivileged and Failsafe DTK development certificates. Certs.xml and .cab are used to provision the DTK development certificates on the device.

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Security\DTK Development Certificates

Security Powertoy

Security Configuration Manager Powertoy for Windows Mobile is a remote configuration tool that runs on a host work station, and enables you to perform the following tasks:

Examine the security configuration of a Windows Mobile device.

Provision one of the Standard Security Configurations onto a mobile device.

Save a mobile device's security configuration.

Add a development certificate to a mobile device.

Sign a file with a Signing Certificate.

Check a file's digital signature.

To install the Security Configuration Manager Powertoy for Windows Mobile, just double-click the Setup.msi installer file.

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Security\Security Powertoy

NOTE: The Security Configuration Manager Powertoy may fail to launch properly on x64-based computers. If you are using Visual Studio 2008, use the included Device Security Manager tool instead. If this is not available, security configuration can be accomplished using XML provisioning.

Security Configuration

.CPFs and related provisioning XML files that allow changing the security policies on the device to one of the standard security configurations.

Default Location:
C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Pocket PC\Security\Security Configuration
C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Smartphone\SecurityConfiguration

RapiConfig

RapiConfig.exe is a desktop configuration tool that allows execution of provisioning XML on a Windows Mobile device or emulator connected using ActiveSync.

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Security\RapiSecurity\RapiConfig

CabSignTool

CabSignTool.exe signs a .cab file and all its executable content (.exe, .dll) with specified certificate(s).

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Security\CabSignTool

Revoke

This tool generates the hash of a certificate or application and outputs an XML document to provision the device to revoke the application or certificate.

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools

Testing

Tool to help developers test applications.

Tool

Description

xHopper

XHopper is a revision of Hopper for Windows Mobile 6.5. Like Hopper, xHopper is a software test tool that simulates input stress on Windows Mobile devices. xHopper stresses all applications made available through the menu system by rapidly sending keystrokes and screen taps in a random fashion. Please see the xHopper Test Tool QuickStart Guide in the Windows Mobile 6.5.3 DTK Help included with this release of the DTK

Default Location: C:\Program Files\Windows Mobile 6.5.3 DTK\Tools\Hopper

Windows Mobile Test Framework

The Windows Mobile Test Framework contains a collection of classes, interfaces, and value types that allow your test code to access controls and resources on a Windows Mobile device.
