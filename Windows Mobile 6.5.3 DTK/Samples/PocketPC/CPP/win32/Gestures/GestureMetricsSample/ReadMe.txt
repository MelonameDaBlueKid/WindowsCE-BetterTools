Code Sample Name: Gesture Metrics Sample

Feature Area: Touch

Description: 
    This sample demonstrates how to use the Gesture APIs.  More specifically, it
    demonstrates:
    - use of TKGetGestureMetrics
    - handling WM_GESTURE GID_HOLD, GID_SELECT and GID_DOUBLESELECT messages
        
    This sample was designed to show how to use TKGetGestureMetrics and interpret
    its results.
    
    This example also shows how to handle GID_HOLD, GID_SELECT and GID_DOUBLESELECT
    messages.

Usage:
    Load the project from Visual Studio 2008, and build and deploy normally.

Relevant APIs/Associated Help Topics: 
    TKGetGestureMetrics
    TKGetGestureInfo
    GESTUREMETRICS
    WM_GESTURE

Requirements: 
    Visual Studio 2008, 
    Windows Mobile 6 Professional SDK,
    WM 6.5 Developer Tool Kit
