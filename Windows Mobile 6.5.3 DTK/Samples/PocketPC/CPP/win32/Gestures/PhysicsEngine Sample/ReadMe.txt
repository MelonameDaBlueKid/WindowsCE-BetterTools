Code Sample Name: Code Sample Name: Physics Engine Sample

Feature Area: Touch

Description: 
    This sample demonstrates how to use the Gesture APIs.  More specifically, it
    demonstrates:
    - use of Physics Engine
    - handling WM_GESTURE GID_PAN, GID_SCROLL and GID_END
        
    This example shows how to handle gesture scrolling and how to properly use 
    Physics Engine.

Usage:
    Load the project from Visual Studio 2008, and build and deploy normally.

Relevant APIs/Associated Help Topics: 
    TKCreatePhysicsEngine
    TKQueryPhysicsEngine
    TKDestroyPhysicsEngine
    TKGetGestureInfo
    PHYSICSENGINESTATE
    PHYSICSENGINEINIT
    WM_GESTURE

Requirements: 
    Visual Studio 2008, 
    Windows Mobile 6 Professional SDK,
    WM 6.5 Developer Tool Kit
