Code Sample Name: FileChangeNotif

Feature Area: 

Description: 
    This sample demonstrates how to listen for changes to the file system.
    
Usage: 
    Load the project from Visual Studio 2005, and build and deploy normally.
    Click on the "av" icon at the bottom of the Today page.  Click Start to
    start listening for file change notifications.  Make a change to the file
    system (for example, create or delete a file), then go back to the sample 
    application and click "Refresh Display" for a report on the changes in the 
    file system. 


Relevant APIs/Associated Help Topics:     
    SHChangeNotifyRegister
    SHChangeNotifyDeregister


Assumptions: 

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Activesync 4.5.

** For more information about this code sample, please see the Windows Mobile 
SDK help system. **
