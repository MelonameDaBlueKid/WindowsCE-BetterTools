Code Sample Name: SpaceWar2D

Feature Area: Bluetooth

Description: 
    This sample is a game that demonstrates how to use the Windows Embedded Source Tools for 
    Bluetooth Technology.


Usage:    
    Load the project from Visual Studio 2005, and build all projects. 
    The SpaceWar project will generate a CAB that you can install on your device.

    Press the "Connect" softkey to choose the device your opponent is using. 
    In order to pick your opponent’s device you should have already established a Bluetooth 
    partnership with that device.


Relevant APIs/Associated Help Topics:    
    System.Drawing
    Microsoft.WindowsMobile.SharedSource.Bluetooth


Assumptions: 
    You have two Windows Mobile devices that have established a Bluetooth partnership.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK,
    .Net compact Framework version 2 or higher,
    Activesync 4.5.
    Windows Mobile 6 devices with Bluetooth.


** For more information about this code sample, please see the Windows Mobile SDK help system. **
