Code Sample Name: OwnerListView

Feature Area: 

Description: 
    This sample demonstrates how to implement an owner drawn ListView control.  
    Also demonstrates how to use various ListView notification events.
    
    Source file listing and summary:
    
    ceownerlistview.cpp - application entry point, owner drawn ListView message 
    handler, window message handler
       
    stdafx.cpp - precompiled type source file
    
    newres.h - helper header for resource script
    
    resource.h - header for resource script
    
    stdafx.h - precompiled header
    
    ceownerlistview.ico
    
    ceownerlistview.rc - resource script 
    
Usage: 
    Load the project from Visual Studio 2005, and build and deploy normally.
    

Relevant APIs/Associated Help Topics:     
    Owner drawn ListView control
    ListView Notification messages.
    Win32 drawing
    GDI objects



Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK,
    Activesync 4.5.

** For more information about this code sample, please see the Windows Mobile 
SDK help system. **
