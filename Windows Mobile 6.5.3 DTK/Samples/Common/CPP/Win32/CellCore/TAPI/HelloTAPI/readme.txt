Code Sample Name: HelloTAPI

Feature Area: Cellcore.

Description:  A simple TAPI dialer. 
    A dialog is presented allowing the user to 
    key in a phone number which is then dialed 
    using the TAPI function tapiRequestMakeCall.

Usage: Run this application from Visual Studio 2005.

Relevant APIs: tapiRequestMakeCall.

Assumptions: The device or emulator has phone capabilities.

Requirements: Visual Studio 2005, Windows Mobile 6 SDK, Activesync 4.5.


** For more information about this code sample, please see the 
Windows Mobile SDK help system. **
