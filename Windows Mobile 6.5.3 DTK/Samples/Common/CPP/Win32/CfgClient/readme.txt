Code Sample Name: CfgClient

Feature Area: Unknown.

Description: 
    CfgClient is a sample application that loads some WAP provisioning XML from 
    a string resource and passes it to DMProcessConfigXML.

    Source file listing and summary are:

        cfgclient.rc - contains the string resource that serves as WAP 
                       provisioning XML.

        main.cpp     - contains the application entry point.

        utils.h      - contains error-checking macros and a string resource ID 
                       definition.

Usage: Run this application from Visual Studio 2005.

Relevant APIs: DMProcessConfigXML.

Assumptions: None.

Requirements: Visual Studio 2005, Windows Mobile 6 SDK,  Activesync 4.5.

Windows Mobile SDK help system. **
