Code Sample Name: Picture Dialer

Feature Area: PIM

Description: Associates a phone number and picture with a digit in a grid.  Pressing the digit will dial the phone number.

Usage: Launch the application, associate a contact and the phone number with a slot, and press the number of the contact to dial the number.

Relevant APIs/Associated Help Topics: POOM, ChooseContact API

Assumptions: None

Requirements:
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK,
    Activesync 4.5.

** For more information about this code sample, please see the Windows Mobile SDK help system. **
