Windows Mobile 6.5 Developer Tool Kit

The Windows Mobile 6.5 Developer Tool Kit adds documentation, sample code, header and library files, emulator images and tools to Visual Studio that let you build applications for Windows Mobile 6.5. This document contains important information about this package. For general information about writing software for Windows Mobile, please see the Windows Mobile Developer Center. The Windows Mobile 6 SDK must also be installed in order to use any of the Windows Mobile 6.5 Gesture API or samples.

Gesture APIs

A new set of APIs is being introduced that will enable application developers to take advantage of the new Windows Mobile 6.5 touch gesture framework. The gesture APIs allow an application to handle touch gesture input and provide a visually consistent experience with the rest of the device UI.

 

The APIs are defined in the following header files:

<gesture.h>

<GesturePhysicsEngine.h>

<WindowAutoGesture.h>

 

Note that the gesture APIs are only available on the Windows Mobile Classic and Professional SKUs. The headers and libraries are installed in the Windows Mobile SDK\Pocket PC\ folder. Samples that make use of these APIs are installed into the Windows Mobile 6.5 Developer Tool Kit\Samples\ folder.

Requirements

The following software is required. Please check each application's system requirements individually.  Windows Mobile 6.0 SDK is required to use the Gesture APIs.

Windows Server 2003

·         Windows Server 2003 Service Pack 2.

·         Microsoft Visual Studio 2005/2008, Standard Edition or above (Express Editions are not supported). SP1 recommended.

·         ActiveSync 4.5. If not currently installed, it is available from Microsoft Windows Mobile web site.

Windows XP

·         Microsoft Windows XP Service Pack 2.

·         Microsoft Visual Studio 2005/2008, Standard Edition or above (Express Editions are not supported). SP1 recommended.

·         ActiveSync 4.5. If not currently installed, it is available from Microsoft Windows Mobile web site.

Windows Vista

·         Microsoft Windows Vista.

·         Microsoft Visual Studio 2005/2008, Standard Edition or above (Express Editions are not supported). SP1 and SP1 Update for Windows Vista required.

·         For synchronizing data, the Windows Mobile Device Center is required. It is available from Microsoft Windows Mobile web site.


Important: "Visual C++ Smart Device Programmability", which is an option of Microsoft Visual Studio 2005/2008 Setup, must be selected and installed during Visual Studio 2005/2008 installation.

Please read and take note of all requirements for those packages, which are detailed on the appropriate download pages.

Development Notes

Compiling Against the Gesture APIs

The Windows Mobile 6.5 gesture APIs are not available on previous versions of Windows Mobile. Any gesture functionality that you utilize in your application will only work on Windows Mobile 6.5 devices.

Gesture API Developer Documentation

More detailed documentation on the gesture APIs is available via the Windows Mobile Developer Center on MSDN.
