Code Sample Name: StateAndNotificationBrokerViewer

Feature Area: State And Notification Broker

Description: 
	This sample demonstrates how to get both state values and state change 
	notifications from the State and Notification Broker.

 
Usage: 
    Load the project from Visual Studio 2005, and build and deploy normally.
    All of the states and their values will be displayed in the top text box.
    (Action on the text box to expand it and see all the values.)
    Use the menu to refresh the top text box.  Any changes to any states 
    will be displayed in the bottom text box.
    

Relevant APIs/Associated Help Topics:     
	Microsft.WindowsMobile.Status


Assumptions: 


Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Standard SDK,
    Activesync 4.5.
