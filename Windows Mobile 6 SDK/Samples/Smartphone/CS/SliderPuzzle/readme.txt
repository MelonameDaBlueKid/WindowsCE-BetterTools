Code Sample Name: SliderPuzzle

Feature Area: Managed Code

Description: 
	SliderPuzzle is a game where the object is to unscramble the tiles.  You 
	can only slide the puzzle pieces into the space that is empty.  

Usage: Build and Run

Relevant APIs/Associated Help Topics: System.Drawing, Key Handling

Assumptions: none

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Standard SDK,
    Activesync 4.5.
