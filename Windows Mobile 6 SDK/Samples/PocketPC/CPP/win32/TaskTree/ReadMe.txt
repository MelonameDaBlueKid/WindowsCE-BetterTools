Code Sample Name: TaskTree.

Feature Area: POOM and POOM custom properties.

Description: Displays Task objects in a tree view. Enables user to add new tasks to the tree.

Usage: build TaskTree.exe and run TaskTree.exe from your device

Relevant APIs/Associated Help Topics: Pocket Outlook Object Model

Assumptions: Application will be run on PocketPC

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK,
    Activesync 4.5.
