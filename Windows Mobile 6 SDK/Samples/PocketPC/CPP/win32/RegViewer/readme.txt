Code Sample Name: RegViewer

Feature Area: Shell

Description: 
    The RegViewer sample is a Pocket PC registry viewer application.
  
Relevant APIs: 
    RegOpenKeyEx
    RegQueryInfoKey
    RegEnumValue
    RegEnumKey
    
Usage:
    Load the project from Visual Studio 2005, and build and deploy normally.

Assumptions: 
    None.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK,
    Activesync 4.5.

** For more information about this code sample, 
please see the Windows Mobile SDK help system. **
