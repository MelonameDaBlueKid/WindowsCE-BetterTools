Code Sample Name: Dynamenu

Feature Area: Shell

Description: 
    The Dynamenu sample demonstrates some techniques for using, getting and 
    setting menu handles on Pocket PC.
  
Relevant APIs: 
    SHGetSubMenu
    SHSetSubMenu
    CheckMenuRadioItem
    CheckMenuItem
    InsertMenu
    DeleteMenu
    

Usage:
   Load the project from Visual Studio 2005, and build and deploy normally.

Assumptions: 
    None.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK,
    Activesync 4.5.

** For more information about this code sample, 
please see the Windows Mobile SDK help system. **
