Code Sample Name: WMPSample

Feature Area: Windows Media Player

Description: 

    This sample shows you how to use the Windows Media Player ATL control 
    from a Win32 application.

    These are the main source files in the project:

    MainWindow.cpp
        Defines the main window class for this application. Its members include 
        the Win32 window and the WMP control.

    WMPSample.cpp
        Defines the main module class for this application.

    WMPEventSink.cpp
        Defines the event dispatcher for this application.


Usage: 
    From Visual Studio, use F5 to build and deploy.
    After the application launches, you can
        - choose the media file using the "Open" menu item.
        - use the Play/Pause softkey as you would in WMP.
        - use the Exit, FullScreen/Mini, Invisible, Stop menu items as you would in WMP.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK.
