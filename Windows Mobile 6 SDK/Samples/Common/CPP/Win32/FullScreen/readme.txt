Code Sample Name: FullScreen

Feature Area: Shell

Description: 
    This sample demonstrates how an application can make full use of the device 
    screen for displaying content using several shell APIs. Using these APIs, 
    the sample application provides a "Full Screen" menu option that allows the 
    taskbar and softkeys to be hidden. Once a softkey button (or the back key 
    on Smarphone) is pressed the taskbar and softkeys are redisplayed.

Usage: 
    Load the project from Visual Studio 2005, and build normally.

    Run the application normally. The sample application has a menu with a 
    "Full Screen" option that can be used to enter full screen mode. 

Relevant APIs/Associated Help Topics:     
    SHFullScreen
    SHCMBM_OVERRIDEKEY message

Assumptions: none.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK
    Activesync 4.5.

SDK help system. **
