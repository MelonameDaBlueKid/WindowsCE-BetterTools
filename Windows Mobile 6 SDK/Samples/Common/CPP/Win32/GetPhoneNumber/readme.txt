Code Sample Name: GetPhoneNumber

Feature Area: Phone

Description: 
    This sample demonstrates how to retrieve the phone number of a device.
    This application does not need to run Trusted, i.e., it does not need 
    to be signed with a certificate from the privileged store.
    
    Source file listing and summary:
    
    getphonenumber.cpp - application entry point, messagebox containing phone 
    					 number if available (see Assumptions)
       
    stdafx.cpp - precompiled type source file
    
    stdafx.h - precompiled header
    
Usage: 
    Load the project from Visual Studio, and build and deploy normally.
    

Relevant APIs/Associated Help Topics:     


Assumptions: 
    If the device uses a SIM, the phone number for the SIM is programmed into it.

Minimum Requirements: 
    Visual Studio 2005
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK

** For more information about this code sample, please see the Windows Mobile 
SDK help system. **
