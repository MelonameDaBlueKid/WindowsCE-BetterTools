Code Sample Name: 
    WDP

Feature Area: 
    Wireless Application Protocol

Description: 
    This sample shows how to use the WAP APIs at the WDP Layer.

Usage: 
    This sample has no UI. It will display information on the Debut Output Window in Visual Studio.
    1. Change the value of g_pszTelNum to your device's phone number.
    2. Build and run the application from Visual Studio 2005.

Relevant APIs/Associated Help Topics: 
    WapOpen
    WapClose
    WapSend
    WapRead
    WapGetNextPrimitiveSize


Assumptions: 
    In order to show what this sample can do, your device must have telephony capabilities.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK,
    Activesync 4.5.
