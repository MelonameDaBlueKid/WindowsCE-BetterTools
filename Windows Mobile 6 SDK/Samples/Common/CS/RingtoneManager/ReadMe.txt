Code Sample Name: Ring Tone Manager

Feature Area: Ring Tones APIs

Description: 
	Ring Tone Manager is an application that scans the device for audio files. It also lets you
    copy those files to a directory where they will be recognize as a ring tone file.
	
Usage: 
    Load the project from Visual Studio 2005, and build and deploy normally. After the program has been
    deployed, press the left soft key to scan the device for media files. Use the menu to preview, stop,
    or copy the ring tone.
	

Relevant APIs/Associated Help Topics: 
        SndGetSoundDirectoriesList
        SndPlaySync
        SndPlayAsync
        SndOpen
        SndStop
        SndClose
        
	
Assumptions: The device is phone enabled.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 6 Professional SDK or
    Windows Mobile 6 Standard SDK,
