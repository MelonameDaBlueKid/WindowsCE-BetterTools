GpsSettings
========
Copyright(c) 2006 Microsoft Corporation 

GpsSettings.exe enables Smartphone users to set up the GPS intermediate driver. 
It works the same as the built-in GPS settings applet on Pocket PC.

Usage: 

Just start GPSSettings.exe and set the Hardware port to the port that receives the GPS data.
If you are using a bluetooth GPS receiver, make sure you pair the GPS device and set it to an
outgoing port. 

For example,
If you choose COM7 as the outgoing port connected to your device on the 
Bluetooth settings, COM7 should be also the hardware port set on GPS settings.

If you set a program port, it will receive a copy of the data streamed to the hardware port.
