Windows Mobile 6 SDK Refresh - Tools

The Windows Mobile 6 SDK Refresh includes tools designed to help developers create applications for Windows Mobile devices.

GPS

Tools to help developers create location-aware applications.

Tool

Description

FakeGPS

FakeGPS enables developers to receive data using the GPS APIs even if there is no GPS receiver on the device. The GPS data is read from NMEA .txt files installed by default in \Program Files\FakeGPS\GPSFiles.

FakeGPS must run in privileged mode because it interacts with the GPS Intermediate Driver and writes to the privileged area of the registry.

FakeGPS ships with a two sample data files:

  • dixies.txt: NMEA files that immediately starts sending a GPS location to the GPS driver.
  • fakegpsdata.txt: Same as dixies.txt, but it takes some time to report a new location.

Default location: C:\Program Files\Windows Mobile 6 SDK\Tools\GPS

GPSSettings

GPSSettings enables Windows Mobile Standard users to configure the GPS intermediate driver and works in the same way as the built-in GPS settings applet on Windows Mobile Pro devices.

To use this tool, start GPSSettings.exe and set the hrdware port to the port that receives the GPS data. If you are using a Bluetooth GPS receiver, make sure you pair the GPS device and set it to an outgoing port.

For example, if you choose COM7 as the outgoing port connected to your device on the Bluetooth settings, COM7 should be also the hardware port set on GPS settings. If you set a program port, it will receive a copy of the data streamed to the hardware port.

Default location: C:\Program Files\Windows Mobile 6 SDK\Tools\GPS

Resolution Awareness

Windows Mobile powered devices are available in a variety of screen resolutions and orientations.

Tool

Description

SetDPI.exe

This tool is used to set the DPI of a bitmap (BMP) file.

Usage: setdpi <filename>.bmp <dpi>

Default location: C:\Program Files\Windows Mobile 6 SDK\Tools\Resolution Aware

HIDPI.exe and RES2EXE.exe

Used to mark a .Net Compact Framework application as resolution aware. For more information, see http://msdn.microsoft.com/library/default.asp?url=/library/en-us/dnppcgen/html/developing_orientation_and_resolution_aware_apps.asp.

Default location: C:\Program Files\Windows Mobile 6 SDK\Tools\Resolution Aware

Security

Several tools and sample security certificates are provided to help developers discover, simulate and predict issues regarding security and their applications.

Tool

Description

RapiSecurity

Consists of .CPF files and related provisioning XML files that allow changing the RAPI policy on a device.

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Security\RapiSecurity

SDK Development Certificates

These files contain .cer, .pfx., .pvk, and .spc files for Privileged, Unprivileged and Failsafe SDK development certificates. Certs.xml and .cab are used to provision the SDK development certificates on the device.

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Security\SDK Development Certificates

Security Powertoy

Security Configuration Manager Powertoy for Windows Mobile is a remote configuration tool that runs on a host work station, and enables you to perform the following tasks:

Examine the security configuration of a Windows Mobile-based device.

Provision one of the Standard Security Configurations onto a mobile device.

Save a mobile device's security configuration.

Add a development certificate to a mobile device.

Sign a file with a Signing Certificate.

Check a file's digital signature.

To install the Security Configuration Manager Powertoy for Windows Mobile, just double-click the Setup.msi installer file.

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Security\Security Powertoy

Security Configuration

.CPFs and related provisioning XML files that allow changing the security policies on the device to one of the standard security configurations.

Default Location:
C:\Program Files\Windows Mobile 6 SDK\Tools\Pocket PC\Security\Security Configuration
C:\Program Files\Windows Mobile 6 SDK\Tools\Smartphone\SecurityConfiguration

RapiConfig

RapiConfig.exe is a desktop configuration tool that allows execution of provisioning XML on a Windows Mobile powered device or emulator connected using ActiveSync.

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Security\RapiSecurity\RapiConfig

CabSignTool

CabSignTool.exe signs a .cab file and all its executable content (.exe, .dll) with specified certificate(s).

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Security\CabSignTool

Revoke

This tool generates the hash of a certificate or application and outputs an XML document to provision the device to revoke the application or certificate.

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools

Testing

Tool to help developers test applications.

Tool

Description

Hopper

Hopper is a software test tool that simulates input stress on Windows Mobile based devices. Hopper will stress all applications made available through the menu system by rapidly sending keystrokes and screen taps in a random fashion.

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Hopper
Windows Mobile Test Framework The Windows Mobile Test Framework contains a collection of classes, interfaces, and value types that allow your test code to access controls and resources on a Windows Mobile device.

Connectivity - Cellular Emulation

Tool to emulate a cellular network

Tool

Description

Cellular Emulator

Cellular Emulator is a software-based emulator to facilitate developers and testers in developing and testing their software under the Windows Mobile platform. The design goal of Cellular Emulator is to replace the radio module in both development and test environments. The advantage of Cellular Emulator is that it provides not only voice but also data connectivity. Moreover, Cellular Emulator is a powerful tool to test various applications under different wireless network conditions in GSM/GPRS and/or UMTS networks. Cellular Emulator replaces FakeRIL on Windows Mobile 6 SDK.

Cellular Emulator can be started from the "Start Menu", "Windows Mobile 6 SDK" and "Tools"

Default Location: C:\Program Files\Windows Mobile 6 SDK\Tools\Cellular Emulator
