      About Tasks

        Use Tasks to keep track of things you need to do. You can use categories to help you organize your tasks and set reminders for them.

          A variety of task information can be displayed on the Today screen.

            If you use Schedule+ or Outlook on your PC, you can synchronize tasks between your device and PC.

             

            Synchronizing Tasks

              Tasks stored on your device can be synchronized with Schedule+ or Outlook on your PC. Task synchronization is automatically selected in ActiveSync.

                New items entered on one computer are copied to the other during synchronization. If you add notes to your tasks, typed notes are added to your Schedule+ or Outlook data, while handwritten notes, drawings, and embedded recordings (if your device supports recordings) are added to Outlook only. Handwritten notes and drawings are saved as metafiles (pictures).

                  For more information on synchronization, see ActiveSync Help on the PC.

                   

                  Why Use Categories?

                    Use categories to group related tasks, contacts, and appointments. Then, use these groupings to quickly display only the information you want. For example, you can assign work-related tasks to the "Business" category and personal items to the "Personal" category. You can then use these categories to filter your tasks, contacts, and appointments and quickly find the information you are looking for.

                      You can also create your own categories, such as "Vacation Planning" to group your trip planning tasks and "Family" to group your relatives' contact information.

                        View Tasks

                          The task list displays a list of your tasks. Overdue tasks are displayed in bold (and in red on color devices). If you have a long list of tasks, tap the sort list (labeled Priority by default) and select the option you want to sort on. To see more task information, tap the task. To see a list of available actions for a task, tap and hold the task. When you complete a task, tap the check box next to it.

                            Tap   > Today to quickly see the number of active tasks you have.

                                Create or change a task

                                  • To quickly create a task, enter text in the Tap here to add a new task box at the top of the screen. If you don't see this box, tap Tools > Enter Bar. When finished, tap OK to return to the task list.
                                  • To create a task with detailed information, such as start and due dates, tap New. To enter notes, tap the Notes tab. When finished, tap OK to return to the task list.
                                  • To change a task, tap it in the task list, and then tap the upper portion of the summary screen. When finished making changes, tap OK to return to the task list.
                                    New tasks are added to the category that is currently displayed.
                                    Tap the Subject arrow in the task details to choose from previous entries.

                                      Set a reminder for a task

                                        1. In the task list, tap the task, and then tap the upper portion of the summary screen.
                                        2. Tap the Due box and set a due date.
                                        3. Tap the Reminder box and select Remind me. You will be reminded at 8:00 in the morning.
                                        4. To change the reminder date, tap the date displayed.
                                          To be automatically reminded of new tasks with a due date, in the task list, tap Tools > Options and select Set reminders for new items.
                                          To choose how you are reminded, for example by a sound, tap   > Settings > Personal tab > Sounds & Reminders.

                                            Add a note to a task

                                              You can add written or typed notes and even recordings (if your device supports recordings) to a task. Notes are handy for maps and drawings.

                                              1. In the task list, tap the task.
                                              2. Tap Tap here to enter notes. For specific instructions, see Notes Help.
                                                To edit an existing note, tap the task in the task list, and then tap the note on the summary screen.

                                                  Make a task recurring

                                                    1. In the task list, tap the task, and then tap the upper portion of the summary screen.
                                                    2. Tap Occurs and select a pattern from the list. Or, to create a new pattern, tap Edit pattern, and follow the steps in the wizard.

                                                      Assign a task to a category

                                                        1. In the task list, tap the task, and then tap the upper portion of the summary screen.
                                                        2. Tap Categories.
                                                        3. Select the categories you want to assign the task to.
                                                        4. To create a new category, tap the Add/Delete tab, enter the category name, and then tap Add. The new category is automatically selected in the Select tab.

                                                          Copy a task

                                                            1. In the task list, tap and hold the task. To select multiple tasks, tap and drag. Then, tap and hold the selected tasks.
                                                            2. On the pop-up menu, tap Create Copy.

                                                              Transfer a task via infrared

                                                                Align the infrared (IR) ports according to the instructions in your owner's manual so that they are unobstructed and within a close range.

                                                                • To send a task, in the task list, tap and hold the task. To select multiple tasks, tap and drag. Then, tap and hold the selected tasks. On the pop-up menu, tap Send via Infrared.
                                                                • To receive a task, in the task list, tap Tools > Receive via Infrared.
                                                                  You can also receive IR transfers by tapping   > Programs > Infrared Receive.

                                                                    Find a task

                                                                      There are two ways to find a task:

                                                                      • Tap   > Find, enter the task name, select Tasks for the type, and then tap Go.
                                                                      • In the task list, tap the category list (labeled All Categories by default), and select the type of task you want displayed. To show all tasks again, select All Categories.
                                                                        In Find, you can tap the Find arrow to select from a list of items you have already looked for.

                                                                          Complete or delete a task

                                                                            • To mark a task completed, in the task list, tap the check box next to the completed task.
                                                                            • To delete a task, in the task list, tap and hold the task. To select multiple tasks, tap and drag. Then, tap and hold the selected tasks. On the pop-up menu, tap Delete Task. The next time you synchronize, the task will be deleted from the PC also.

                                                                              Change Options

                                                                                In the task list, tap Tools > Options.

                                                                                • To see the start and due dates for a task in the task list, select Show start and due date.
                                                                                • If you are having trouble reading the tasks in the task list, select Use large font. You will see fewer tasks at a time.
