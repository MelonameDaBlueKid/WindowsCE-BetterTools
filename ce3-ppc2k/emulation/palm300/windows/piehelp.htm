      About Pocket Internet Explorer

        You can use Pocket Internet Explorer to view Web pages on your device in two ways: by downloading Web pages through synchronization or by
        connecting to the Internet.

         

        About Favorites

          If you are using Microsoft Internet Explorer 4.0 on the PC, you can synchronize favorite links. If you are using Microsoft Internet Explorer 5 on the PC, you can also synchronize mobile favorites. Synchronizing favorite links enables you to have the URLs of your favorite Web sites at your fingertips. Synchronizing a mobile favorite enables you to view Web content on your device while you are disconnected from your Internet service provider and PC.

            Favorites are stored in the Mobile Favorites subfolder of Favorites in Internet Explorer on the PC. This folder was created when you selected the Favorites information type for synchronization in ActiveSync on your PC. Favorites must be stored in this folder to be synchronized.

              About Channels

                You can synchronize two types of channels with your mobile device: mobile channels and AvantGo channels.

                  A mobile channel is a small-scale Web page designed specifically for offline viewing on a Windows-powered device. You subscribe to mobile channels from a regular Web site via Internet Explorer on your desktop computer. The mobile channels are stored in the Mobile Favorites subfolder of Favorites in Internet Explorer. This folder was created when you installed ActiveSync on your PC. Then, synchronize your device with your desktop computer to download content.

                    AvantGo is a free interactive service that gives you access to personalized content and thousands of popular Web sites. Unlike mobile channels, you can subscribe to AvantGo channels directly from your device. Then, you synchronize your device and desktop computer or connect to the Internet to download the content. For more information, visit http://avantgo.com/help.

                        Synchronize a favorite link

                          1. In ActiveSync on your PC, click Tools > Options, and select Favorite. For more information on using ActiveSync, see ActiveSync Help.
                          2. In Internet Explorer on your PC, save or move favorite links to the Mobile Favorites subfolder in the Favorites list. For more information on using Internet Explorer, see Internet Explorer Help on the PC.
                          3. Connect your mobile device to your PC. If synchronization does not start automatically, in ActiveSync, click File > Synchronize.
                            If you add or delete a favorite link in the Mobile Favorites folder on your device, the next time you synchronize, the same change is made on the PC.

                              Synchronize a mobile favorite

                                1. In ActiveSync on your PC, click Tools > Options, and select Favorite. For more information on using ActiveSync, see ActiveSync Help.
                                2. In Internet Explorer 5, browse to the Web page you want to view offline.
                                3. Click Tools > Create Mobile Favorite.
                                4. To change the link name, enter a new name in the Name box.
                                5. In the Update box, select a desired update schedule to keep the Web page in the Mobile Favorites folder up to date. You can also update content by clicking Tools > Synchronize in Internet Explorer.
                                6. To save the link in a subfolder of Mobile Favorites, click Create In and select the desired subfolder.
                                7. Click OK. Internet Explorer will download the latest version of the Web page.
                                8. To download the pages that are linked to the mobile favorite you just created, in Internet Explorer on the desktop computer, right-click the mobile favorite and then click Properties. On the Download tab, specify the number of links deep you want to download. To conserve device memory, only go one level deep.
                                9. Connect your mobile device to your PC. If synchronization between your device and PC does not start, in ActiveSync, click File > Synchronize.

                                  Synchronize a mobile channel

                                    1. In ActiveSync on your PC, click Tools > ActiveSync Options, and select Favorite. For more information on using ActiveSync, see ActiveSync Help.
                                    2. To see available mobile channels, in Internet Explorer 4.0, visit the Windows CE Mobile Channels List Web page (http://www.microsoft.com/windowsce/channels).
                                    3. On the Web page, click Add Mobile Channel to download the channel to your PC.
                                    4. In the Add Favorite dialog box, select Make available offline.
                                    5. Unlike Favorites, you do not need to set up a schedule to receive updated content for the mobile channel, nor do you need to synchronize Internet Explorer with the Internet. The mobile channel's content provider will download any updated information to your PC each time you connect to the Internet. The next time you synchronize, the channel will be transferred to your device.

                                      Synchronize an AvantGo channel
                                        1. Add the Mobile Application Link information type to ActiveSync by installing it from the AvantGo folder in the Extras folder on the ActiveSync CD.
                                        2. In Pocket Internet Explorer on your device, tap  to display your list of favorites.
                                        3. Tap the AvantGo Channels link.
                                        4. Tap Activate.
                                        5. Follow the directions on the screen. You will need to synchronize your device with your desktop computer and then tap My Channels to complete the AvantGo setup.
                                        6. When synchronization is complete, tap the AvantGo Channels link in your list of favorites to see a few of the most popular channels.
                                        To add or remove channels, tap the Add or Remove link.

                                                    Delete a favorite link

                                                      1. Tap .
                                                      2. Tap the Add/Delete tab and then select the link you want to delete.
                                                      3. Tap Delete.
                                                        Only the link to the Web page is deleted, not the Web page content stored on your device. To delete the Web page itself, tap Tools > Options > General tab > Delete Files.

                                                          Connect to the Internet

                                                            To view Web pages that are not stored on your device (offline favorites), you must set up a remote connection to your ISP or network. For more information, see Connections Help.

                                                              Once connected, view Web pages in the following ways:

                                                              • To view your favorite Web pages, tap   and then the Web page you want to view.
                                                              • To visit a Web page that is not in your Favorites list, enter the address in the address bar at the top of the screen and then tap Go. If you don't see the address bar, tap View > Address Bar.
                                                                If you are having problems connecting, you may need to access the Internet using a proxy server. Ask your network administrator for the necessary information and enter it in Tools > Options > Connections tab and select Access remote content automatically.
                                                                You can have Pocket Internet Explorer automatically connect to the Internet when you attempt to access a page that is not stored on your device by specifying a connection in Tools > Options > Connections tab.

                                                                      View the History List

                                                                        Tap View > History.

                                                                        • To view a Web page, tap it.
                                                                        • To change the information displayed in the History list, tap the sort list (labeled Page Title by default).
                                                                        • To change the number of days of links stored in the History list or to clear the History list, tap Tools > Options > General tab.

                                                                                    Fit the Web page to the screen

                                                                                      By default, Pocket Internet Explorer attempts to arrange the Web page so that you can see all content without scrolling horizontally. In most cases, you will only need to scroll vertically to see content. To turn this feature off, tap View > Fit to Screen.

                                                                                        Change options

                                                                                          Tap Tools > Options.

                                                                                          • To free storage space, in the General tab, tap Delete Files. All Web pages stored on your device, including offline favorites content, will be deleted.
                                                                                          • To speed up page loading time, in the Advanced tab, clear options under Media.

                                                                                            Change security options

                                                                                              Pocket Internet Explorer supports the security protocols used by secure Web pages. When accessing a secure Web page, you can send personal information safely. To determine if a Web page is secure, tap View > Properties.

                                                                                                To change security settings, tap Tools > Options > Advanced tab.

                                                                                                  Cookies contain information about your identity and preferences so that a Web page can tailor information to your needs. The Web page sends the file, and it is stored on your mobile device. You can prevent Pocket Internet Explorer from accepting any cookies by clearing Allow cookies.
