        Buttons

          Tap   > Settings > Personal tab > Buttons.

          • On the Program Buttons tab, customize the program hardware buttons to open your most used programs. Under Button, tap the button you want to assign, and select a new program in Button assignment.
          • If scrolling is too fast or slow, adjust the delay and repeat rates on the Up/Down Control tab.

           

          Input

            Tap   > Settings > Personal tab > Input.

            • On the Input Method tab, select an input method and change desired options.
            • On the Word Completion tab, select desired options for word completion.
            • On the Options tab, select desired options for writing and recording (your device may not support recording). These options apply to any program on the device where you can record or write. These options do not apply to text you enter using the input panel.
              You can also change input options by tapping the   arrow and then Options.
              Mobile Voice is a 2.4 KB audio codec (compressor decompressor). It is the recommended format because it provides good voice recording and takes far less storage space than PCM (Pulse Code Modulation), which provides slightly better sound quality. However, PCM recordings may take up to 50 times more storage space than Mobile Voice recordings. PCM cannot be used to create recordings that are embedded in notes.

                Menus

                  Tap   > Settings > Personal tab > Menus.

                  • On the Start Menu tab, select the items you want to appear under .
                  • On the New Menu tab, you can turn on the New button pop-up menu. Once turned on, tap the New button arrow to see a list of information types you can create.
                    The New button in specific programs opens new items for the corresponding program only.
                    You can add subfolders and shortcuts to the Start menu. Using ActiveSync on the PC, open the Start Menu folder located in the Windows folder on your device. Create folders and shortcuts as you normally do on the PC and store them in this folder.

                       

                      Owner Information

                        Tap   > Settings > Personal tab > Owner Information.

                        • In the Identification tab, enter your personal information. Select Show information when device is turned on so that if you lose your device, it can be returned to you.
                        • In the Notes tab, enter any additional text you want to display when the device is turned on, such as "Reward if found."

                         

                        Password

                          Tap   > Settings > Personal tab > Password.

                          • Using the keypad, enter a 4-digit password. Once you tap OK and save this change, you will need to enter this password to access this setting again.
                          • To require a password for accessing information on the device, select Require password when device is turned on.

                          Caution If you forget your password, you must follow the directions in your owner's manual to clear memory before you can access your device. This erases all files and data you have created, and programs you have installed. If you've used ActiveSync to create a backup file, all of your program data can be restored. For more information on creating a backup file, see ActiveSync Help.

                           

                          Sounds & Reminders

                            Tap   > Settings > Personal tab > Sounds & Reminders.

                            • In the Volume tab, select the types of actions you want to hear sounds for. If you don't want to hear sounds, move the System volume slider to Silent.
                            • In the Sounds tab, you can customize the sounds you hear for different events. (Your device may not support custom sounds.) Tap the event name you want to change and then select a new sound from the Sound list.
                            • In the Reminders tab, select how you want to be reminded of upcoming appointments and tasks.
                              Turning off sounds and the notification light helps conserve battery power.
                              To quickly turn off all device sounds, tap   > Today and then the speaker icon in the lower-right corner of the screen.

                               

                              Today

                                Tap   > Settings > Personal tab > Today.

                                • Select the information you want displayed on the Today screen.
                                • To further customize the information, select an information type, and then tap Options (not available for all information types).

                                 

                                About

                                  Tap   > Settings > System tab > About.

                                  • The Version tab displays important device information, including the amount of memory installed.
                                  • In the Device ID tab, you may need to change the device name if you are trying to connect to a network and cannot because another device with the same name is already connected.

                                   

                                  Align Screen

                                    Tap   > Settings > System tab > Align Screen.

                                    • When aligning the touch screen, be sure to tap the target firmly and accurately.
                                    • You can also use hardware buttons to align the touch screen. For more information, see your owner's manual.

                                     

                                    Backlight

                                      Tap   > Settings > System tab > Backlight.

                                      • You will see this setting only if your device has a backlight that can be customized.
                                      • Using the backlight while on battery power substantially reduces battery life.
                                        To quickly turn the backlight on or off, tap   > Today and then the lightbulb icon in the lower-right corner of the screen.

                                         

                                        Clock

                                          Tap   > Settings > System tab > Clock.

                                          • If you visit a particular city often, select it as your Visiting city in the Time tab so that you can quickly see the correct date and time.
                                          • You can use your device as a travel alarm clock by setting a wake-up alarm in the Alarms tab. Tap the alarm icon to select the type of alarm you want, such as a sound.
                                          • To change the way the date or time is displayed on your device, go to Regional Settings.
                                            The time is displayed on the navigation bar. Tap and hold the time to see today's date.

                                             

                                            Contrast

                                              Tap   > Settings > System tab > Contrast.

                                              • You will see this setting only if your device supports adjusting the screen contrast.
                                              • Turning on the backlight can also make the screen easier to read.

                                               

                                              Memory

                                                Tap   > Settings > System tab > Memory.

                                                • For information about freeing memory, see Free Memory.
                                                • The Running Programs tab displays a list of active programs on your device. You can manually start and stop a program by selecting it in the list and tapping the appropriate button. You may need to do this if a program becomes unstable or program memory is low.

                                                 

                                                Power

                                                  Tap   > Settings > System tab > Power.

                                                  • The amount of remaining operating time varies, depending on battery type and how you use your device.
                                                  • To conserve battery power, set the device to turn off after 3 minutes or less.
                                                  • Use the AC adapter to plug your device into external power whenever possible, especially when using a CompactFlash card, using a modem or other peripheral, using the device backlight, connecting to a PC, or letting the notification light flash for extended time periods.

                                                   

                                                  Regional Settings

                                                    Tap   > Settings > System tab > Regional Settings.

                                                    • On the Region tab, select your region to set the default number formats.
                                                    • To change the time display to 24 hours, tap the Time tab and choose a time style that starts with a capital "H."
                                                    • To further customize, tap the tabs and select desired options. The characteristics available are determined by the region selected on the Region tab.

                                                     

                                                    Remove Programs

                                                      Tap   > Settings > System tab > Remove Programs.

                                                      • You can remove programs that you installed in storage memory only. Programs that come with your device cannot be removed.
                                                      • You can quickly return your device to the factory default configuration by clearing all memory. All data will be lost. For instructions, see your owner's manual.
