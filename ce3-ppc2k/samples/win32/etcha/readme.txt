

========================================================================
       WIN32 APPLICATION : Etcha
========================================================================

Name: Etcha
============



What is it: 
============
Turns the Pocket PC screen into an electronic8 drawing pad.


Features: 
==========

Allows the developer to quickly build a free-hand drawing pad for the Pocket PC.

This application includes one button which, if clicked once, will clear the screen,
and if double-clicked, will exit the application.



Demonstrates:
==============

Basic Win32 APIs.
