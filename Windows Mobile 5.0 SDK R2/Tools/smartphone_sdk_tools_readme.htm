Additional Tools and XML Samples

This directory contains several additional tools and sample XML files, some of which are not described in the SDK documentation.

 

Details on the tools will appear in future releases.

 

 

Cabsigner.exe

This tool signs the contents of a CAB file and then resigning the CAB file with the same certificate.  For more information refer to cabSigner_Readme.htm in the SDK tools folder.

CabWiz.ddf

Template directive file to create CAB files

 CabwizSP.exe

This is a special version of the CabWiz tool that supports the additional security aspects of Smartphone 2002 .CAB files. This tool is documented in the SDK documentation. (Look in the Index under “CabWizSP”.)

EmuASCfg.exe

This tool configures ActiveSync and the Pocket PC emulator to allow establishment of an ActiveSync partnership over the Ethernet Virtual Switch.

You must use eMbedded Visual C++ to boot the Pocket PC emulator using the VirtualSwitch option before running this tool.  For usage information, type “emuascfg.exe /?” at the command line.

Extract.exe

This tool allows you to selectively extract files out of a .CAB file. For usage information, type “extract /?” at the command line.

Makecert.exe

This tool is also available on the Microsoft Developer Network (MSDN). It provides a quick way to create certificate files. For usage information, type “makecert /?” at the command line or consult MSDN.

RapiConfig.exe

RapiConfig provides a way to cause configuration XML to be executed on a device or emulator connected via ActiveSync. For usage information, type “rapiconfig /?” at the command line.

Revoke.exe

This tool generates provisioning XML files for revoking a file or certificate on the device. For usage information, type “revoke /?” at the command line.

Signcode.exe

This tool is also available on the Microsoft Developer Network (MSDN). It allows you to sign a file with a certificate. For usage information, type “signcode /?” at the command line or consult MSDN.

Spdps.exe

This tool prepares the various development packages (groups of files used for debugging or runtime support) for use in the emulator or on a real device. This is documented in the SDK documentation. (Look in the Index under “development file signing tool”)

 

AddCert.xml

Sample XML to add a certificate to the certificate store.

PolicyAdd.xml

Sample XML to add or change a security policy.

PolicyQuery.xml

Sample XML to query a security policy. Note: Configuration Manager queries can only succeed if run locally on the device using DMProcessConfigXML. They do not function via Over-The-Air configuration.

QueryCert.xml

Sample XML to query a certificate from the certificate store. Note: Configuration Manager queries can only succeed if run locally on the device using DMProcessConfigXML. They do not function via Over-The-Air configuration.

QueryCertParms.xml

Sample XML to query the base64 encoded certificate and role mask individually of a certificate. Note: Configuration Manager queries can only succeed if run locally on the device using DMProcessConfigXML. They do not function via Over-The-Air configuration.

QueryStore.xml

Sample XML to query the certificate store. Note: Configuration Manager queries can only succeed if run locally on the device using DMProcessConfigXML. They do not function via Over-The-Air configuration.

RemoveCert.xml

Sample XML to remove a certificate from the store.

RevAdd.xml

Sample XML to add a hash to the revocation list.

RevHashQuery.xml

Sample XML to query if a particular hash is in the revocation list. Note: Configuration Manager queries can only succeed if run locally on the device using DMProcessConfigXML. They do not function via Over-The-Air configuration.

RevListQuery.xml

Sample XML to query the revoaction hash list. Note: Configuration Manager queries can only succeed if run locally on the device using DMProcessConfigXML. They do not function via Over-The-Air configuration.
