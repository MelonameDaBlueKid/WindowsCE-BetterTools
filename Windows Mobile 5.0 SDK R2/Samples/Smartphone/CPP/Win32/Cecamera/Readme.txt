Code Sample Name: CECamera

Feature Area: Camera

Description: Take pictures and videos using Camera native API.

Usage: Specify the options in the dialog, then press "Start" to launch Camera 
       for taking picture or video.

Relevant APIs/Associated Help Topics: SHCameraCapture

Assumptions: You already have a PocketPC or Smartphone with Camera.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 Pocket PC SDK or
    Windows Mobile 5.0 Smartphone SDK
    ActiveSync 4.0.


** For more information about this code sample, please see the Windows Mobile SDK help system. **
