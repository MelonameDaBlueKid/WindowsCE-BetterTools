Code Sample Name: POOMTasks

Feature Area: Pocket Outlook

Description: 
	Shows how to use POOM to create, manage, and delete tasks.

	poomdata.cpp contains most of the code that deals with interacting
	with POOM for this application.  It stores and manipulates the
	poom model.  

	poomtasks.cpp controls most of the windows and dialogs.  It contains
	the window procs and initializers.

Usage: Build and Run.

Relevant APIs/Associated Help Topics: 
	IPOutlookItemCollection, IRecurrencePattern, ITask

Assumptions: none.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 Smartphone SDK,
    ActiveSync 4.0.

** For more information about this code sample, please see the Windows Mobile SDK help system. **
