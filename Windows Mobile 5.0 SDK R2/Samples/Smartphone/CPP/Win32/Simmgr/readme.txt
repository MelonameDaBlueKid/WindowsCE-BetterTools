Code Sample Name: SIMMgr

Feature Area: Phone

Description: 
	Sample that shows how to add and delete phone number entries on your SIM.

Usage: Build and Run

Relevant APIs/Associated Help Topics: 
	SimInitialize, SimGetDevCaps, SimDeinitialize, SimReadPhonebookEntry

Assumptions: You have a SIM in your device or use the emulator

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 Smartphone SDK,
    ActiveSync 4.0.

** For more information about this code sample, please see the Windows Mobile SDK help system. **
