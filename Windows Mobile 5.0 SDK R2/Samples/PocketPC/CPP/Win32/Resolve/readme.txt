Code Sample Name: Resolve

Feature Area: Shell

Description: 
    The Resolve sample demonstrates how to resolve a host name 
    (e.g. www.microsoft.com) into an IP address, or an IP address into a host 
    name. This sample uses the gethostbyaddr and gethostbyname functions from 
    the WinSock API. 

Relevant APIs: 
    WinSock API.
   
Usage:
    Load the project from Visual Studio 2005, and build and deploy normally.

    To use this sample, simply enter a host name or an IP address in the upper 
    edit control and tap either the "IP to Host" or "Host to IP" button to 
    display the result in the lower edit control.

Assumptions: 
    This sample operates in the emulator or on the real device. (Accessing 
    Internet addresses may require that you enter your proxy information 
    in Connection Manager.)
    
Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 Pocket PC SDK,
    ActiveSync 4.0.

** For more information about this code sample, 
please see the Windows Mobile SDK help system. **
