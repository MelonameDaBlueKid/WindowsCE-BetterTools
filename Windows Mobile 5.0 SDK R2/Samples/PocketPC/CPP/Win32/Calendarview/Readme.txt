Code Sample Name: CalendarView

Feature Area: POOM (Pocket Outlook Object Model)

Description: 
    The sample illustrates how to use the POOM IItem interface for appointment 
    items, and POOM notifications. It displays a listview with all 
    calendar items.

Usage: 
    Load the project from Visual Studio 2005, and build and deploy normally.

Assumptions: You will need to have some Calendar items. 

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 Pocket PC SDK,
    ActiveSync 4.0.

** For more information about this code sample, 
please see the Windows Mobile SDK help system. **
