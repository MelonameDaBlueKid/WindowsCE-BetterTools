Code Sample Name: ProjectEnumeration

Feature Area: Shell

Description: 
    ProjectEnumeration is a sample that demonstrates the Project document 
    and file manipulation API. It consists of four dialogs and a small number 
    of functions for file manipulation. The dialogs provide users with a set of 
    user interface elements for opening and saving documents.
    
    The documents are organized by projects. A file is associated with its project 
    folder and identified by its file name. It has a set of editable attributes 
    that describes various file properties, such as the  location, type, size 
    and date of the last modification. The four dialogs are: "Folder", "Open",
    "Save As" and "Properties". This sample demonstrates how to open these 
    four dialogs.
  
Relevant APIs: 
    Project document and file manipulation API
   
Usage:
    Load the project from Visual Studio 2005, and build and deploy normally.

    This application focuses on the Demonstration of Projects Technology and 
    leaves users to finish up the code for opening projects, opening files 
    and saving files.

Assumptions: 
    None.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 Pocket PC SDK,
    ActiveSync 4.0.
