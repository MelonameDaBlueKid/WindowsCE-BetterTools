
Code Sample Name: BithdaysToCalendar
Feature Area: Managed Windows Mobile APIs

Description: Iterates thorugh the entire contact list and adds the Birthdays of the each of the contacts to Calendar as a yearly recurring all day event

Usage: 
    Run the sample. From menu, choose Add to add the Birthdays to Calendar.

    Note:
    This sample works for both Windows Mobile 5.0 SmartPhone SDK and Windows Mobile 5.0 Pocket PC SDK.
    If you need to pick a specific configuration:
        1. Right click on the project.
        2. Click on "Change Target Platform"
        3. Choose the platform you decide to target and click OK.

    Load the project from Visual Studio 2005, and build and deploy normally.



Relevant APIs/Associated Help Topics: 
    Microsoft.WindowsMobile.PocketOutlook.Appointment
    Microsoft.WindowsMobile.PocketOutlook.AppointmentRecurrence


Assumptions: Need to have contacts with valid birthdays in the contact list.

Requirements: 
    Visual Studio 2005, 
    Windows Mobile 5.0 SmartPhone SDK.
