Code Sample Name: SmsIm

Feature Area: SMS (Short Message Service)

Description: 
	This app illustrates how to use SMS interception.  It provides a
	UI for managing multiple instant messaging conversations over SMS.


Usage: 
    Load the project from Visual Studio 2005, and build normally.
   
Relevant APIs/Associated Help Topics: 
    Microsoft.WindowsMobile.PocketOutlook
    Microsoft.WindowsMobile.PocketOutlook.MessageInterception
    
Assumptions: The device has SMS support. 

Requirements: 
	Visual Studio 2005, 
	Windows Mobile 5.0 Pocket PC SDK or  
	ActiveSync 4.0.

** For more information about this code sample, please see the Windows Mobile 
SDK help system. **
